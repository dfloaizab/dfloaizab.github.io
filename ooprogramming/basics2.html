Fundamentos de Programación Orientada a Objetos

Resumen conceptual y técnico con ejemplos prácticos y código en Java

Clase

Una clase es una estructura conceptual y técnica que define un tipo de objeto dentro de un programa. Describe las propiedades que tendrán sus instancias mediante atributos, y las acciones que podrán realizar mediante métodos.

Puede entenderse como un molde o plantilla a partir del cual se construyen objetos concretos durante la ejecución del programa.

Ejemplo conceptual

La idea de “Automóvil” describe características generales como color o velocidad. Cada automóvil específico será un objeto derivado de esta clase.

Ejemplo adicional

La clase “Libro” puede contener atributos como título, autor y número de páginas, así como métodos como abrir o cerrar.

Ejemplo en Java

class Automovil {

    String color;
    int velocidad;

    void acelerar(){
        velocidad += 10;
    }

    void frenar(){
        velocidad -= 10;
    }

}

Segundo ejemplo en Java

class Libro {

    String titulo;
    String autor;

    void abrir(){
        System.out.println("El libro ha sido abierto");
    }

}

Objeto

Un objeto es una instancia concreta de una clase creada durante la ejecución del programa. Cada objeto posee valores específicos en sus atributos y puede ejecutar los métodos definidos por su clase.

Ejemplo conceptual

Un automóvil rojo de marca Toyota constituye una instancia específica de la clase Automóvil.

Ejemplo adicional

Un libro concreto titulado "Cien años de soledad" con un autor y un número de páginas es un objeto derivado de la clase Libro.

Ejemplo en Java

Automovil auto1 = new Automovil();

auto1.color = "Rojo";
auto1.velocidad = 60;

Segundo ejemplo en Java

Libro libro1 = new Libro();

libro1.titulo = "Cien años de soledad";
libro1.autor = "Gabriel García Márquez";

Encapsulamiento

El encapsulamiento consiste en ocultar los detalles internos de un objeto y restringir el acceso directo a sus datos. La interacción con el objeto se realiza mediante métodos controlados que garantizan la integridad del estado interno.

Ejemplo de la vida real

Un cajero automático permite retirar dinero mediante una interfaz sencilla sin exponer el sistema bancario interno.

Ejemplo adicional

Un teléfono móvil permite realizar llamadas mediante botones o pantalla táctil sin que el usuario manipule directamente los circuitos internos.

Ejemplo en Java

class CuentaBancaria {

    private double saldo;

    public void depositar(double cantidad){
        saldo += cantidad;
    }

    public double consultarSaldo(){
        return saldo;
    }

}

Segundo ejemplo en Java

class Persona {

    private int edad;

    public void setEdad(int edad){
        if(edad > 0){
            this.edad = edad;
        }
    }

    public int getEdad(){
        return edad;
    }

}

Abstracción

La abstracción consiste en representar únicamente las características esenciales de un objeto y ocultar los detalles innecesarios para el contexto del problema.

Ejemplo de la vida real

Un control remoto permite manejar un televisor mediante pocos botones, ocultando la complejidad electrónica interna.

Ejemplo adicional

Un mapa del metro representa estaciones y conexiones sin mostrar todos los edificios de la ciudad.

Ejemplo en Java

abstract class Animal {

    abstract void hacerSonido();

}

Segundo ejemplo en Java

abstract class Vehiculo {

    abstract void mover();

}

Herencia

La herencia permite que una clase derive de otra, reutilizando atributos y métodos ya definidos. La clase derivada puede extender o modificar ese comportamiento.

Ejemplo de la vida real

Un perro es un tipo particular de animal y comparte características comunes con otros animales.

Ejemplo adicional

Un profesor y un estudiante pueden considerarse tipos específicos de persona.

Ejemplo en Java

class Animal {

    void respirar(){
        System.out.println("Respirando");
    }

}

class Perro extends Animal {

    void ladrar(){
        System.out.println("Guau");
    }

}

Segundo ejemplo en Java

class Persona {

    String nombre;

}

class Estudiante extends Persona {

    String carrera;

}

Polimorfismo

El polimorfismo permite que distintos objetos respondan de manera diferente a una misma operación.

Ejemplo de la vida real

El verbo “moverse” se manifiesta de forma distinta en un automóvil, una persona o un pez.

Ejemplo adicional

El método emitir sonido se manifiesta de manera distinta en perros, gatos y aves.

Ejemplo en Java

class Animal {

    void sonido(){
        System.out.println("Sonido generico");
    }

}

class Perro extends Animal {

    void sonido(){
        System.out.println("Guau");
    }

}

Segundo ejemplo en Java

class Gato extends Animal {

    void sonido(){
        System.out.println("Miau");
    }

}
