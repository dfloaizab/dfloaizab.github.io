Principios de Programación Orientada a Objetos

Abstracción

La abstracción consiste en modelar entidades del mundo real destacando únicamente sus características esenciales, ocultando detalles innecesarios.

Ejemplo 1

class Vehiculo: def mover(self): pass

Ejemplo 2

class CuentaBancaria: def depositar(self, monto): pass

Ejemplo 3

class Figura: def area(self): pass

Encapsulamiento

El encapsulamiento protege los datos internos de un objeto y controla su acceso.

Ejemplo 1

class Persona: def __init__(self): self.__edad = 0 def set_edad(self, edad): self.__edad = edad

Ejemplo 2

class Cuenta: def __init__(self): self.__saldo = 0 def depositar(self, monto): self.__saldo += monto

Ejemplo 3

class Producto: def __init__(self, precio): self._precio = precio

Herencia

La herencia permite que una clase derive de otra reutilizando su comportamiento.

Ejemplo 1

class Animal: def hablar(self): pass class Perro(Animal): def hablar(self): return "Guau"

Ejemplo 2

class Vehiculo: pass class Auto(Vehiculo): pass

Ejemplo 3

class Figura: pass class Circulo(Figura): pass

Polimorfismo

El polimorfismo permite usar una misma interfaz con diferentes implementaciones.

Ejemplo 1

def hablar(animal): print(animal.hablar())

Ejemplo 2

class Gato: def hablar(self): return "Miau"

Ejemplo 3

class Pato: def hablar(self): return "Cuac"
